INNOVA FUTURA

WHERE KIDS LEARN CODING




Innova Futura is an online coding class currently serving ages 8 - 13. It was created by LogiPhillia. We plan to teach HTML and CSS, languages used for creating websites, to those who want to learn it. Learning with us is completely free. All teachers at Innova Futura are people younger than 18 years old that want to teach other children the basics of programming, but we do so in a friendlier, less boring way than many other programming courses. We know that people learn best face-to-face, but we also believe that all people should have a right to learn to code. To solve this, we use video calls to teach. We also have a specific way of teaching.
